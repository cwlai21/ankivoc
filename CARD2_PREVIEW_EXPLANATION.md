### 🔍 问题根源

您在**卡片浏览器预览区**看到的 "example" 和 "sample" 是 **Anki 内置的占位符文本**，用于演示 `{{type:}}` 字段的功能。

Card 2 使用 `{{type:中文}}` 创建拼写输入框。这个字段在不同模式下的表现：

| 模式 | 显示内容 |
|------|----------|
| **实际学习** | 空白输入框，等待您输入 |
| **提交答案后** | "您的答案 ↓ 正确答案" 对比 |
| **预览模式** | Anki 显示示例 "sample ↓ example" |

### 🧪 验证步骤

请在 Anki 中实际学习这张卡片（不是预览）：

1. 点击 **学习** 按钮开始复习
2. 当 Card 2 出现时：
   - **Listening 模式**：播放音频，显示空白输入框
   - **Reading 模式**：显示英文解释，显示空白输入框
3. 输入框应该是**空白的**，没有 "example" 占位符
4. 输入答案（如：手術室）并提交
5. 背面显示：您输入的答案 vs 正确答案

### ❓ 如果实际学习时仍有问题

如果您在**实际学习**（不是预览）时仍然看到 "example"/"sample"：

#### 可能原因 A：Anki 版本问题
某些 Anki 版本的 `{{type:}}` 字段渲染有bug。

**解决方法**：
```
1. 检查 Anki 版本（帮助 → 关于 Anki）
2. 如果版本 < 2.1.50，建议更新到最新版本
3. 更新后重启 Anki
```

#### 可能原因 B：缓存问题
Anki 可能缓存了旧的模板数据。

**解决方法**：
```
1. 工具 → 管理笔记类型
2. 选择"中文-(R/L)"
3. 点击"选项" → "重新生成所有卡片"
4. 关闭并重启 Anki
```

#### 可能原因 C：字段值有误
虽然我们检查过数据，但可能有边缘情况。

**解决方法**：
```
1. 打开卡片浏览器
2. 找到有问题的卡片
3. 点击"字段"按钮，检查所有字段内容
4. 确认"中文"字段包含正确的中文文字（如：手術室）
5. 如果发现任何字段包含 "example"，删除它
```

### 📝 如果只是预览问题

如果问题**仅出现在预览中**，而实际学习时正常：

这是 Anki 的正常行为，无需修改。预览模式下，Anki 使用占位符演示 `{{type:}}` 字段的功能。

### 🛠️ 可选解决方案：移除拼写功能

如果您不需要拼写练习功能，可以移除 `{{type:}}` 字段：

**修改方法**：
1. 打开 [regenerate_all_templates.py](backend/regenerate_all_templates.py)
2. 找到 Card 2 Front 模板中的：
   ```html
   {{^No Spell}}
   <div class="Text_Card radius">
   <div class="Text-answer">{{type:中文}}</div>
   </div>
   {{/No Spell}}
   ```
3. 改为直接显示答案：
   ```html
   {{^No Spell}}
   <div class="Text_Card radius">
   <div class="Text-answer">{{中文}}</div>
   </div>
   {{/No Spell}}
   ```
4. 运行 `python regenerate_all_templates.py` 更新模板
5. 在 Anki 中同步模板

但这样会失去拼写练习功能。
